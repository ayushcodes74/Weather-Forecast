# 🌦️ Weather Forecast App

A simple and responsive Weather Forecast web application built using **HTML, CSS, and JavaScript**. It fetches real-time weather data from the **OpenWeather API** and displays the current weather of any city entered by the user.

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather description
- 💧 Humidity information
- 🌬️ Wind speed
- 🖼️ Dynamic weather icons
- ✅ Input validation
- ⚠️ Error handling for invalid city names
- 📱 Fully responsive design
